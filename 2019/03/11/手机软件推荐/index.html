手机软件推荐

发表于 2019-03-11 | 更新于 2019-03-23 | 分类于 软件推荐

　　随着手机配置发展的越来越高，各类毒瘤软件也似乎找到了温床，广告推送，后台驻留等一系列问题在国内应用生态圈已经泛滥。今天给大家介绍一些良心且好用的软件，这些软件的开发者基本上都是个人开发，个人建议有能力者入正以支持开发者，我不会在本站发布其相关破解版。

简介：

—–全新酷安V9—–

● 多样新首页
频道入口更清晰，想看哪个看哪个

● 手机吧来了
机友走心干货，酷友真实点评

● 收藏大升级
优质内容，任性分组

● 发布会直播
和百万酷友一起看直播，聊新机

● 应用备份单
一键备份应用列表，再不怕丢失

②纯纯写作

简介：

令人感到安心，顺滑流畅的写作体验，如你所愿，这就是纯纯写作。我们希望写作能够回到原本的样子：快速、纯粹、随时、有安全感、具备良好的写作体验。
　　纯纯写作实现了在 Android 上完美的行间距和段间距，它能够使您的文本看起来更加舒适，清晰。同时，它还实现了「快捷输入栏」和「格式化工具」，以及许多喜人的细节内容。
　　最重要的，在使用这个编辑器时，它将会保证您的内容永远不会丢失，除非您主动将它清空。否则即使误操作将文本删除，或瞬间断电，您仍然能够从历史记录或备份中将它恢复。
　　纯纯写作还在流畅体验上下了很多功夫，它拥有 完美的键盘响应逻辑。使用纯纯写作时，你能够往下拖动快捷输入栏来收起软键盘；能够点击文章某一行后，动态获得合适的光标焦点定位；能够在收起软键盘时，文章位置保持不动 … 许多这样的细节内容，当你对比使用其他编辑器项目时，你才会发现纯纯写作做得更加出色、流畅，细致入微。
　　蕴繁于简。许多编辑器该有的功能，纯纯写作也都没有落下，比如 WebDAV 云备份、段首缩进、生成长图片、指纹加锁、Markdown、更换信纸、夜间模式、单词联想输入、打字机模式、撤销、字数统计 等等。但即使这样，纯纯写作仍然保持了简约的设计风格，遵循 Material Design 设计，并且它还有一个像是哆啦 A 梦 时光机 的图标，寓意着「文字能够连接过去，遥想未来」，也照应了纯纯写作特别提供的「历史记录」功能。
　　您能够最快速度到达灵感页面，能够随时随地中断和继续写作，这些我们都已经为您做好了。
　　令人感到安心，顺滑流畅的写作体验，如你所愿，这就是纯纯写作，请享受写作吧！

笔者：

非常良心的写作软件，开发者人特别好，如果你是谷歌用户或者需要多设备使用，建议用兑换码在谷歌商店购买兑换，支付宝购买时联系作者说明情况就可以了。

③阅读

简介：

开源的阅读软件来了：如今的小说阅读软件总是在不断的添加广告，作为一个程序猿这是受不了的，于是开源的阅读软件来,你不用再担心广告。
本软件fork一个无人维护的阅读软件，经过大量修改，实现自定义书源，添加大量自定义功能，我的阅读我做主。

笔者：

优秀的阅读软件，作者为了版权纠纷移除了内置书源，不过在关于页面有公众号，里面有上千的书源提供下载。

④Share

简介：

Share，第三方新浪微博客户端；
简约的Material设计，找回刷微博的乐趣，各种惊喜的细节等待你一一发现；
兼容Android版本：5.0+；
支持的功能：
微博编辑、评论、点赞、收藏、删除；
查看热门评论、查看评论图片、评论点赞；
管理微博分组（排序、新增、编辑、删除）;
查看热门话题、微博、热搜；
观看微博视频；
好友私信；
发微博支持多张图片，上传视频，选择地点，发送范围；
高级版说明：
打开应用设置即可看到开通入口，高级版订阅30RMB/年，功能如下：

  • 隐私设置-微博关键字屏蔽
  • 主题风格-自定义主题颜色
  • 发表内容时切换账号
  • 登陆账号上限提高到30个
  • 悬浮窗视频支持迷你模式

⑤白描

简介：

随着用户越来越多，白描的研发与运营成本越来越高，计划于3月1日起调价，黄金会员将从25元改价为26元，普通会员将从6元调价为8元，感谢您的支持与理解。依然是一次购买永久有效。
白描是一款准确高效的 OCR 文字识别与扫描软件，识别准确度高，速度快，扫描文件清晰，可生成PDF。
拍照翻译文字、公司文件转录、阅读笔记摘录、截图文字提取、身份证识别、购物小票识别，纸质文件电子化存档等多场景适用。
● 批量识别。
同类领跑者，一次识别50张图片。
● 表格识别，自动识别生成Excel。
● 自由导出与编辑识别结果。
● 识别结果自动存为历史记录，检索方便。
● 识别结果翻译
支持简体中文、日语、英语、韩语、法语、西班牙语、阿拉伯语、俄语、德语、葡萄牙语、意大利语、繁体中文、粤语、文言文的互译；
● 多语言识别
支持中文、英语、日语、韩语、法语、德语、俄语、西班牙语的识别；留学生说“白描用来处理上课笔记实在是太好用了”。
● 自动识别文档边界生成扫描件，多种文档色彩可供选择。
● 自动识连续拍照扫描件，可生成PDF
● 灵活选择
● 快速校对
在识别结果页面点击“校对”按钮进行校对，弥补OCR技术的不完美之处。
● 识别结果自动分段，弥补了大多数识别软件无法还原原文段落的问题。

⑥Z直播

简介：

一个APP就可以看多个平台直播~
目前直播平台众多，主播们与不同的平台签约，导致用户往往需要装多个直播客户端。而且这些官方客户端往往非常臃肿，各种杂七杂八的内容让用户无法专心看直播，大多数情况下视频和弹幕就够了。
「Z直播」是我业余时间开发的一款聚合直播应用，在写这个 APP 过程中我做了非常多的优化，整个应用也非常纯净。

⑦储存空间清理

简介：

【存储空间清理】帮你解决文件过多、存储不足和卡顿！

存储空间不足？“其他”文件占用好几G？试试存储空间清理，解决所有存储不足的问题。

【核心功能】

★1.分析每个文件(夹)的信息（体积大小、对系统影响、来源等），一目了然；

★2.一键替换：可将目录一键替换为空文件，来尝试阻止目录再生成（尝试阻止广告或缓存生成）。

★3.一键整理APK安装包：规范重命名，整理到统一目录中；

★4.一键清理空文件夹，同时还可清理空白文件；

★5.专清功能：微信清理、QQ清理、微博清理等，所有软件的垃圾无所遁形；

★6.重复文件：手机中有超多相同的重复文件，找出来，清理掉；

★7.强大的定时清理功能，可定期自动清理指定的文件(夹)；

★8.超级搜索：强大到爆表的多功能文件搜索器；

★9.最新文件：找出手机中每天最新产生的文件；

★10.最旧文件：找出藏在手机中最久的文件；

★11.疑似缓存：找出极可能是缓存垃圾的目录；

★12.强大的文件备注，备注“不要清理”还能保护文件；

【软件特点】

对存储文件全面分析，让原本杂乱无章的文件，从此一目了然。自己手动清理，也将信手拈来。

在用过同类的那些清理大师、优化大师、清理助手后，使用本软件也依然能再次清理出大量空间。

注意：文件过多很容易导致耗电和卡顿，减少文件占用，很有必要！

本软件纯净无广告！手机优化加速，值得一试。

预知更多功能，还望您亲自探索！

笔者：

像OPPO，VIVO，小米，华为等主流大厂手机就不用了，手机自带的清理已经够用了，如果你是类原生系统的话用着还挺不错，一般情况下我都是手动清理，用这个偶尔扫描一下大文件和重复文件，高级版也就一瓶水钱，而且绑定手机，推荐使用。

⑦IDM+

简介：

此版本为谷歌版去除谷歌授权验证，签名不同需卸载原版安装。

怎么样设置软件语言为中文呢，点击左上角/Sittings/General settings/Language选择Chinese，然后重启。很简单，给第一次上手使用的人看的。谷歌原版已完整汉化。

如有侵权联系删除。In case of infringement, contact deleted.

IDM +是最快，最先进的下载管理器（通过Torrent下载支持）适用于Android。它高达比正常下载速度更快500％。 它不运行后台服务，如果没有什么下载和智能下载选项被禁用这增加了电池的寿命。

阅读常见问题解答@ http://www.apps2sd.info/idmp/faq

教程@ https://www.youtube.com/watch?v=4VotpvLnTrg
